# Changelog

## [0.3.1] - 2026-03-30
### Added
- Expanded `parallelMerge` to accept a custom merge function for more flexible result aggregation.

---

## [0.3.0] - 2026-03-30
### Changed
- Updated `package.json` metadata to better surface examples and versioning.
- Enhanced `README` and examples with clearer Node.js version requirements.
- Switched versioning guidance to reference `orinocoflow_version`.

---

## [0.2.0] - 2026-03-17
### Changed
- Removed the Bun runtime requirement to simplify usage in standard Node.js environments.
- Updated the example script to match the new runtime assumptions.

---

## [0.1.0] - 2026-03-12
### Added
- Initial `orinocoflow` library including core orchestration primitives, tests, examples, and documentation.
- Node-compatible CLI and dual build outputs for both library and CLI usage.
- Support for interrupt/resume flows, retry limits, and Bun source export in early workflows.
- Project branding assets including logo and initial README setup for npm publish.
